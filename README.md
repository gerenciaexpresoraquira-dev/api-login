EVIDENCIA AA5-EV01

Servicio web para registro y autenticación de usuarios.

Tecnologías utilizadas:
- Node.js
- Express
- CORS
- Body-parser

Funcionalidades:
- Registro de usuario con usuario y contraseña
- Inicio de sesión con validación
- Respuesta de autenticación exitosa o error

Endpoints:
POST /registro
POST /login
